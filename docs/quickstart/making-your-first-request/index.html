Making your first request

Creating the Server object​

To use the erlc-js module, you need to make a Server object for your private server.

main.js
const { serverKey } = require("./config.json") // Get the server key from our config.json file that we made earlier
const { Server } = require("erlc-js") // Get the erlc-js module

const privateServer = new Server(serverKey) // Create a Server object with our server key (you can also parse an authorization key along with this)

Initiating the server​

It isn't required to initiate your Server before using it, but it's best practice and strongly recommended to do so.

Initiating your Server sets it up with all of the information it needs and verifies that the server actually exists.

main.js
privateServer.initiate() // Initiate the Server

If you add this on to the rest of the code, it'll show you a Server initiation message in the console (hopefully!).

Well, technically you've just made your first request. But you can't really do much with it, so now we'll see what we can do.

Getting the server info​

Now you've initiated the Server, all the server information should be there. Here's how to get the server info:

main.js
privateServer.initiate().then(() => { // Initiate the server, but wait for it to finish before executing
    console.log(`Server initiated as ${privateServer.name}.`) // Print the Server's name
    console.log(`Join me with the code ${privateServer.joinCode}`) // Print the Server's join code
})

Sending your first command​

Now you have your Server initiated, most of the work's already done! All of the functions in the Server should now work.

One of those methods are .sendCommand(), which is probably the main method you'll be using.

The syntax of this command is as follows:

privateServer.sendCommand("command", ["array","of","command", "arguments"])

Here's how you can send a 'Hello World!' message to your private server.

privateServer.sendCommand("m", ["Hello World!"])

Well, you've done it! You've successfully sent your first request to the server. Pat yourself on the back, it's not an easy feat!

If you've done this all correctly, this should be your final script:

main.js
const { serverKey } = require("./config.json") // Get the server key from our config.json file that we made earlier
const { Server } = require("erlc-js") // Get the erlc-js module

const privateServer = new Server(serverKey) // Create a Server object with our server key (you can also parse an authorization key along with this)
privateServer.initiate() // Initiate the Server
privateServer.sendCommand("m", ["Hello World!"]) // Send the command
